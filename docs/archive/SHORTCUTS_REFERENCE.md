# AMK Command Center - Keyboard Shortcuts Reference

Quick reference guide for all keyboard shortcuts in the application.

## How to Access

Press `⌘K` (Mac) / `Ctrl+K` (Windows/Linux) or `?` anywhere in the app to see the shortcuts modal.

---

## Global

| Shortcut    | Action                       |
| ----------- | ---------------------------- |
| `⌘K` or `?` | Show keyboard shortcuts help |
| `Esc`       | Close modals/dialogs         |
| `⌘⇧Z`       | Undo last action             |

---

## Tasks

| Shortcut | Action                                |
| -------- | ------------------------------------- |
| `Space`  | Toggle task completion (when focused) |
| `P`      | Pause task (when focused)             |
| `N`      | New task                              |
| `E`      | Edit task (when focused)              |
| `D`      | Delete task (when focused)            |

---

## Navigation

| Shortcut  | Action                 |
| --------- | ---------------------- |
| `⌘1-5`    | Switch between tabs    |
| `↑` / `↓` | Navigate through tasks |

---

## Voice

| Shortcut | Action                 |
| -------- | ---------------------- |
| `⌘Enter` | Start voice recording  |
| `Esc`    | Cancel voice recording |

---

## Search

| Shortcut | Action             |
| -------- | ------------------ |
| `⌘F`     | Focus search box   |
| `⌘⇧F`    | Open global search |

---

## Modal

| Shortcut | Action       |
| -------- | ------------ |
| `⌘↵`     | Save/Submit  |
| `Esc`    | Cancel/Close |

---

## Audio

| Shortcut | Action                |
| -------- | --------------------- |
| `⌘M`     | Toggle audio feedback |

---

## Tips

1. **Help is always available**: Press `⌘K` or `?` anytime to see all shortcuts
2. **Context-aware**: Some shortcuts only work when elements are focused
3. **Escape works everywhere**: `Esc` closes modals and cancels operations
4. **Mac vs Windows**: `⌘` (Command) on Mac = `Ctrl` on Windows/Linux

---

**Power User Tip**: Learn 3-5 shortcuts you use most often. You'll save hours over time!
